# GeneratepromptAi

## Dashboard UI Library - مكتبة واجهات التحكم

مكتبة واجهات تحكم جاهزة للاستخدام في مشاريع Laravel، توفر تصميم عصري ومتجاوب مع جميع أحجام الشاشات.

### البنية التنظيمية

```
GeneratepromptAi/
├── public/
│   └── dashboard-ui/
│       ├── css/
│       │   └── dashboard.css      # ملف الأنماط الرئيسي
│       └── js/
│           └── dashboard.js      # ملف JavaScript الرئيسي
├── resources/
│   └── views/
│       ├── layouts/
│       │   └── app.blade.php     # ملف التخطيط الأساسي
│       └── dashboard/
│           └── example.blade.php # مثال على الاستخدام
├── DASHBOARD_UI_README.md        # دليل الاستخدام الكامل
└── README.md                     # هذا الملف
```

### البدء السريع

1. استخدم ملف `resources/views/layouts/app.blade.php` كتخطيط أساسي لصفحاتك
2. راجع ملف `DASHBOARD_UI_README.md` للتعليمات التفصيلية
3. شاهد مثال الاستخدام في `resources/views/dashboard/example.blade.php`

### المميزات

- ✅ تصميم عصري وجذاب
- ✅ متجاوب بالكامل مع جميع الأجهزة
- ✅ قائمة جانبية قابلة للطي
- ✅ نظام إشعارات مدمج
- ✅ سهولة التكامل والتخصيص
